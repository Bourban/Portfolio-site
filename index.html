Hello there! I'm Jordan Walsh; game programmer, designer and (occasionally) tech support for fellow team members. If you've stumbled across this site you're likely interested in some of my work:

  • The Big Up, a 2D defense game created by a small group of postgrad students which included myself and one other programmer. In addition to my contribution to the codebase I also created and maintained the programming task list as well as trying to keep us as organised as possible.
  • My ongoing C++ SFML solo projects, which can be found on my GitHub. In particular I'd like to highlight the implementation of the Command pattern, which encapsulates user input and allows for easy character swapping.
  • My occasional short projects, like Snek, a console-based Snake remake in C++. Most of my short projects are open source, so feel free to have a peek.

If you'd like to contact me feel free to send me an email, tweet me at @BourbanMola or connect with me on LinkedIn.
